
### **Option 2: Use a Simple HTTP Server**

```bash
# Python 3
cd frontend
python -m http.server 8000

# Then visit: http://localhost:8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (if you have it)
npx http-server
```

### **Option 3: Use GitHub Pages (Deploy)**

To host your frontend on GitHub Pages:

1. Go to your repository settings: https://github.com/NAVEED261/Cloud-native-AI-400-PROJECT-1/settings
2. Scroll to "GitHub Pages" section
3. Select "Deploy from a branch"
4. Choose "master" branch and "/root" directory (if root has index.html) or create a special gh-pages branch
5. Your site will be published at: `https://NAVEED261.github.io/Cloud-native-AI-400-PROJECT-1/`

### **Option 4: Use Online Viewers**

You can paste the raw HTML URL into online HTML viewers:
- https://htmlpreview.github.io/?[RAW_URL_HERE]
- https://raw.githack.com/[path_to_file]

Example:
- https://htmlpreview.github.io/?https://raw.githubusercontent.com/NAVEED261/Cloud-native-AI-400-PROJECT-1/master/frontend/index.html

```bash

```

**Vercel:**
